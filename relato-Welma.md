## Relato de Dificuldades
Durante o desenvolvimento da Etapa 3 do projeto, as principais dificuldades enfrentadas foram:
1. **Correção da Estrutura HTML:** O código inicial apresentava diversas tags mal fechadas e erros de sintaxe que prejudicavam a renderização correta das páginas.
2. **Implementação do Carrossel:** Criar um carrossel de destaques utilizando apenas JavaScript puro e CSS, garantindo uma transição suave entre os itens.
3. **Padronização Visual:** Unificar o design de todas as páginas (Login, Dashboard, Gerenciamento) através de um único arquivo de estilos (`style.css`), garantindo consistência visual e facilidade de manutenção.
4. **Navegação:** Ajustar os links internos e renomear arquivos para garantir que a navegação entre o portal público e a área administrativa funcionasse sem erros.
5. **Feedback Interativo:** Implementar lógica de busca que fornece feedback visual imediato (mudança de cor das bordas) conforme o usuário digita, o que exigiu um cuidado extra na manipulação do DOM.

## Tecnologias Utilizadas

### 1. HTML5
- **Justificativa:** Utilizado para a estruturação de todo o conteúdo do portal. É a base fundamental da web moderna, permitindo a criação de páginas acessíveis e bem organizadas.

### 2. CSS3
- **Justificativa:** Utilizado para toda a parte visual do projeto. Foram implementadas animações de *fade-in* para suavizar a transição entre as páginas, efeitos de *hover* em botões e links para melhor interatividade, e um design moderno com gradientes e sombras.

### 3. JavaScript (ES6+)
- **Justificativa:** Utilizado para adicionar interatividade ao portal. Responsável pela lógica do carrossel de destaques na página inicial e pela simulação do processo de login e recuperação de senha, proporcionando uma experiência de usuário mais dinâmica.
